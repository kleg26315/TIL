변수와 자료형

변수 선언

함수 내부에서 'var 변수명;'으로 선언하면 지역변수가 됨
함수 내부에서 '변수명;'으로 선언하거나 함수 밖에서 '변수명;' 혹은 'var 변수명;'으로 선언하면 전역변수가 됨
전역변수와 지역변수가 동일한 이름인 경우 함수 내부에서 변수 명을 호출하면 지역 변수가 우선권을 가짐
전역 변수 사용 시 'window.변수 명' 혹은 'this.변수 명'으로 표현하여 지역 변수와 구분
전역 변수는 해당 window내에서 어디서든지 사용할 수 있으며, 지역 변수는 해당 함수 내에서만 사용 가능

자료형

자바스크립트에서는 자료형 별로 변수 타입이 지정되지 않고 리터럴에 의해 자료형 결정



데이터 형변환

문자열과 숫자의 + 연산

문자열 + 숫자 = 문자열
숫자 + 숫자 + 문자열 = 숫자 먼저 계산되고 그 뒤의 문자열과 합쳐짐



typeof()연산자

선언 시 자료형을 지정하지 않기 때문에 변수 명만으로도 데이터의 자료형을 확인할 수 없음
변수의 자료형 확인 시 자주 사용



강제형변환

Number(), parseInt(), parseFloat()함수를 이용해 강제 형변환
