스크립트 개요

스크립트 언어

기본 프로그램 동작을 사용자의 요구에 맞게 수행되도록 하는 용도로 사용
매우 빠르게 배우고 짧은 소스코드 파일로 상호작용하도록 고안되었음

자바스크립트

웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체지향 프로그래밍 언어

자바스크립트 사용법

1. 태그에 직접 간단한 소스코드를 작성해서 실행하는 방법

on이벤트명= "실행할 함수명([전달값]);"
2. script태그에 작성하여 실행하는 방법
3. 별도의 js파일로 작성해서 가져다 사용하는 방법

태그에 직접 자바스크립트 코드를 작성하여 실행

script태그 안에 함수 단위로 소스 작성하여 실행

별도의 js파일로 작성해 실행

console.log()로 개발자 도구 콘솔 화면에 출력
